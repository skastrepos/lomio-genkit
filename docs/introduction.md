# Introduction / 介绍

Lomio GenKit helps AI SaaS developers reach a visible AI result quickly, then grow the project into a commercial application with clear package boundaries.

Lomio GenKit 帮助 AI SaaS 开发者快速看到可运行的 AI 结果，并通过清晰的包边界逐步扩展为可商业化应用。

## Goal / 目标

From `lomio create` to your first AI result in under 5 minutes.

从 `lomio create` 到第一个 AI 结果，目标控制在 5 分钟以内。

## Design Direction / 设计方向

- Start with a working demo provider.
- Keep provider, database, config, and shared code separated.
- Make the web app useful before adding platform complexity.

- 从可工作的 Demo Provider 开始。
- 保持 Provider、数据库、配置和共享代码边界清晰。
- 在加入平台复杂度之前，先让 Web 应用真正可用。
