# Quick Start / 快速开始

## Requirements / 环境要求

- Node.js 20.11 or newer.
- pnpm 9 or newer.

- Node.js 20.11 或更高版本。
- pnpm 9 或更高版本。

## Install / 安装

```bash
pnpm install
```

## Run / 运行

```bash
pnpm dev
```

Open `http://localhost:3000`.

打开 `http://localhost:3000`。

## Generate a Story / 生成故事

Enter one sentence in the AI Story Generator and click `Generate`. The app calls the local API route, which uses the Demo AI Provider and returns a simulated story.

在 AI Story Generator 中输入一句话并点击 `Generate`。应用会调用本地 API Route，使用 Demo AI Provider 返回一段模拟故事。

## CLI Preview / CLI 预览

```bash
pnpm cli -- create my-ai-app
```

This command is reserved in V0.1 and does not scaffold files yet.

该命令在 V0.1 中只是预留结构，暂不生成文件。
