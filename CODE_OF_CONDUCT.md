# Code of Conduct / 行为准则

Lomio GenKit is built for a respectful, practical, and welcoming open-source community.

Lomio GenKit 致力于建设尊重、务实、友好的开源社区。

## Expected Behavior / 鼓励的行为

- Be kind, direct, and constructive.
- Assume good intent while still naming problems clearly.
- Respect different levels of experience.
- Keep discussions focused on the project and its users.

- 保持友善、直接、建设性。
- 默认他人善意，同时清楚指出问题。
- 尊重不同经验水平的参与者。
- 让讨论聚焦项目和用户。

## Unacceptable Behavior / 不接受的行为

- Harassment, insults, threats, or discrimination.
- Publishing private information without consent.
- Repeatedly derailing technical discussions.

- 骚扰、侮辱、威胁或歧视。
- 未经同意公开他人隐私信息。
- 反复偏离技术讨论主题。

## Enforcement / 执行

Maintainers may remove comments, close issues, or limit participation when behavior harms the community.

当行为伤害社区时，维护者可以删除评论、关闭 issue 或限制参与。
