# Contributing / 贡献指南

Thank you for considering a contribution to Lomio GenKit.

感谢你考虑为 Lomio GenKit 做贡献。

## Principles / 原则

- Keep the first AI result path fast and understandable.
- Prefer small, well-scoped pull requests.
- Avoid adding commercial platform features before their roadmap phase.

- 保持第一个 AI 结果路径快速、清晰。
- 优先提交范围明确的小 PR。
- 不要在路线图阶段之前加入商业平台型功能。

## Local Development / 本地开发

```bash
pnpm install
pnpm dev
pnpm typecheck
```

## Pull Requests / Pull Request

Before opening a PR, please make sure:

- The change matches the V0.1 scope or an accepted roadmap item.
- Documentation is updated in both English and Chinese when user-facing behavior changes.
- Type checks pass.

提交 PR 前请确认：

- 变更符合 V0.1 范围，或对应已接受的路线图条目。
- 用户可见行为变化时，文档已同时更新英文和中文。
- 类型检查通过。
