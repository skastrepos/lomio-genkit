# Security Policy / 安全政策

## Supported Versions / 支持版本

V0.1 is an early skeleton release. Security fixes will target the latest main branch until stable releases begin.

V0.1 是早期骨架版本。在稳定版本开始前，安全修复将优先面向最新 main 分支。

## Reporting a Vulnerability / 报告漏洞

Please do not open public issues for sensitive security reports. Email the maintainers or use the repository security advisory flow when available.

请不要为敏感安全问题创建公开 issue。请通过维护者邮箱或仓库安全公告流程报告。

Include:

- Affected package or app.
- Reproduction steps.
- Impact and suggested severity.
- Any known workaround.

请包含：

- 受影响的包或应用。
- 复现步骤。
- 影响范围和建议严重程度。
- 已知临时规避方案。
